## What's New in 1.1.0

- Added a new `Circle` drawing tool.
- Circle sizing uses **diameter**.
- Circle measurement shows **diameter** and **area**.
- Added copy/paste support for circles and labels.
- Added multi-select and lasso group move for faster layout edits.
- Added auto-save for create/edit/move/resize/delete/toggle actions.
- Improved polygon right-angle workflow:
  - stricter right-angle indicator
  - keybinding-assisted right-angle snapping while drawing
- Improved rectangle vertex behavior to preserve rectangle geometry.
- Improved area label placement stability.
- Reordered toolbar for faster access to paired actions:
  - Select + Pan
  - Length toggle + Show/Unshow Length
  - Area toggle + Show/Unshow Area
  - Copy + Paste
  - Undo + Redo
